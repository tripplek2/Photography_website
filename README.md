# Photography_website

My project is all about my photography and it all entails photos that i took with a litle biography of me.

## Folder structure

- index.html-main page
- about.html-biography information
- style.css-stylesheet
- images-images used.
- LICENSE-usage rights of the project
- README.md-provides information about my project

## Installation

1. clone the repository:

  git clone https://github.com/tripplek2/Photography_website.git 

2. Navigate to the project directory:

  cd Photography_website and then open the project
